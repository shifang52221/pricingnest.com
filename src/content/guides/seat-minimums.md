## Quick checklist
- Define a minimum seat floor per tier.
- Align seat minimums with onboarding cost.
- Use seat minimums to reduce churn risk.
- Review seat minimums quarterly.
- Document exceptions clearly.

## Step-by-step
1. Estimate support and onboarding cost per account.
2. Set a minimum seat count to cover costs.
3. Compare seat minimums to usage-based alternatives.
4. Model ARPA impact by segment.
5. Update pricing page with clear seat rules.

## What to watch
- Low minimums allow unprofitable accounts.
- High minimums reduce SMB conversion.
- Hidden minimums create disputes.

## Common mistakes
- Using the same minimum for all segments.
- Ignoring ramp periods for new accounts.
- Treating minimums as temporary discounts.
