# Reports

This folder is for human-readable review artifacts that can be archived,
compared month over month, and used for internal records.

## Report Types

- `pricingnest-summary-YYYY-MM-DD.md`
  - short executive summary
  - one-sentence verdict
  - top findings
  - priority actions

- `pricingnest-audit-appendix-YYYY-MM-DD.md`
  - detailed evidence log
  - Search Console tables
  - crawl findings
  - manual review notes
  - trust and template-risk observations

## Operating Rule

- Keep the summary short and decision-oriented.
- Keep the appendix evidence-heavy and traceable.
- Match every conclusion to a source file, crawl result, or page review.
- Use the same date in the file name and the audit folder whenever possible.

## Related Folder

- `audits/`
  - raw exports, crawl output, and structured review material
